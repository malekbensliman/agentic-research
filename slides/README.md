# slides/

Slide content, **one markdown file per module**, grouped by session:

- `session-1/` — Integrating AI tools into the research workflow
- `session-2/` — AI for academic research
- `assets/` — images referenced by slides

**Markdown is the source of truth.** Decks (`.pptx`) are rendered from these files (see `../scripts/`) and are not committed — edit the `.md`, then render.

Each module file follows a shared template (to be finalized): front-matter (title, duration, learning goals) followed by per-slide sections with body content, speaker notes, and image notes.
